Descubra o Agrinho!

O Projeto Agrinho, em parceria com a Alura, é uma iniciativa incrível para levar o mundo da programação e do pensamento computacional para as crianças. Através de atividades lúdicas e divertidas, como o Scratch, os pequenos aprendizes descobrem como criar suas próprias histórias, jogos e animações!

O que é Programação?

Programar é como dar instruções para o computador fazer o que você quer! É como montar um quebra-cabeça gigante, onde cada peça é uma instrução. Com a programação, você pode criar desde um joguinho simples até um aplicativo superlegal para o celular.

No Agrinho, você aprende a programar de um jeito muito visual e intuitivo, usando bloquinhos de código no Scratch. É divertido e fácil!

Seus Primeiros Projetos!

Com o Agrinho, você pode:

  • Criar animações com personagens e cenários.
  • Desenvolver joguinhos simples com regras que você inventa.
  • Contar histórias interativas onde o leitor decide o que acontece.
  • Soltar a imaginação e construir o que quiser!
